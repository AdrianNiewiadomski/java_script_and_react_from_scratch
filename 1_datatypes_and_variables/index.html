Datatypes and variables

To store our data in memory we may create variables. To do this we use a keyword "var", then a variable name in camelCase.

JavaScript
var myVariable;

If we want to assign a value to variable we use an assignment operator "=".

JavaScript
var myVariable = 2; console.log(myVariable);
Console
2

In JavaScript variables may hold various data types. Some of the data types are boolean values, numbers, strings, and objects.

JavaScript
var isPasswordCorrect = true; console.log(isPasswordCorrect);
var myString = 'Mary had a little lamb'; var myString = 'Mary had a little lamb' + 'and a horse'; console.log(myString);
var myDoc = document; console.log(myDoc);
Console

true

Mary had a little lamb

HTMLDocument

JavaScript is a dynamically-typed language. This in practice means that type of data you store in variable may change.

JavaScript
var x = 2; console.log(x + ' is ' + typeof(x)); x = 'Mary had a little lamb'; console.log(x + ' is ' + typeof(x)); x = true; console.log(x + ' is ' + typeof(x)); x = document; console.log<(x + ' is ' + typeof(x));
Console

2 is number

Mary had a little lamb is string

true is boolean

[object HTMLDocument] is object
